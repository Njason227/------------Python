## 1. Вход в систему

1. Откройте браузер и перейдите по адресу приложения
2. На главной странице нажмите **«Войти в систему»**
3. Введите имя пользователя и пароль
4. Нажмите **«Войти»**

> При первом входе используйте учётную запись администратора, созданную при развёртывании.

### Учётные записи по умолчанию

| Логин | Пароль | Роль |
|-------|--------|------|
| `admin` | `admin123` | Администратор |

> **Важно:** при первом входе обязательно смените пароль администратора.

## 2. Панель управления

Доступна по ссылке **«Панель»** в верхнем меню.

### Разделы панели

| Раздел | Описание |
|---|---|
| **Всего пациентов** | Общее число зарегистрированных пациентов |
| **Ожидают** | Пациенты, ещё не распределённые в отделение |
| **Распределены** | Пациенты, назначенные в отделение |
| **Госпитализированы** | Пациенты, завершившие процесс приёмного отделения |
| **Поступления** | Количество поступлений за сегодня / неделю / месяц |
| **Загрузка отделений** | Процент занятости коек по каждому отделению |
| **По полу / возрасту / тяжести** | Статистическое распределение пациентов |

## 3. Работа администратора

### 3.1. Управление пользователями

**Просмотр списка пользователей:**
- Перейдите в меню **«Администрирование → Пользователи»**

**Создание нового пользователя:**
1. Нажмите **«Новый пользователь»**
2. Заполните поля:
   - Имя пользователя (логин)
   - Фамилия, Имя
   - Роль (Врач приёмного / Старшая медсестра / Администратор)
   - Телефон
   - Пароль и подтверждение
3. Нажмите **«Создать»**

### 3.2. Управление справочниками

Администратор имеет доступ ко всем справочникам через меню **«Справочники»**:

#### Пол
- Просмотр, добавление, редактирование, удаление половых категорий

#### Тяжесть психического состояния
- Уровни от 1 (Лёгкая) до 4 (Крайне тяжёлая)

#### Тяжесть физического состояния
- Уровни от 1 (Удовлетворительное) до 4 (Критическое)

#### Диагнозы МКБ-11
- Поиск по коду или названию
- Добавление новых诊断нозов с указанием кода, названия, блока и класса МКБ

#### Отделения
- Название отделения
- Профиль (описание диагнозов)
- Ограничение по полу (Мужское / Женское / Смешанное)
- Возрастная категория (Детское / Взрослое / Любой)
- Количество коек (общее и занятые)

#### Правила распределения
- Привязка отделения к диагнозу МКБ-11
- Ограничения по полу, возрасту, тяжести состояния
- Приоритет правила (1 — высший, 5 — низший)

### 3.3. Администрирование через Django Admin

```
https://ваш-домен.ru/admin/
```

## 4. Работа врача приёмного отделения

### 4.1. Регистрация нового пациента

1. Перейдите в раздел **«Пациенты»** → **«Зарегистрировать пациента»**
2. Заполните форму:
   - **Фамилия, Имя, Отчество** — полное ФИО пациента
   - **Дата рождения** — автоматически рассчитывается возраст и категория (до 18 / 18+)
   - **Пол** — Мужской / Женский
   - **Диагноз (МКБ-11)** — выбрать из справочника диагнозов
   - **Тяжесть психического состояния** — Лёгкая / Умеренная / Тяжёлая / Крайне тяжёлая
   - **Тяжесть физического состояния** — Удовлетворительное / Средней тяжести / Тяжёлое / Критическое
3. Нажмите **«Сохранить»**
4. Пациент появится в списке со статусом **«Ожидает распределения»**

### 4.2. Просмотр карточки пациента

Нажмите на ФИО пациента в списке. Карточка содержит:
- Персональные данные (ФИО, возраст, пол)
- Медицинские данные (диагноз, тяжесть)
- Информацию о распределении (отделение, врач, дата)
- Историю распределений

### 4.3. Редактирование данных пациента

1. Откройте карточку пациента
2. Нажмите **«Редактировать»**
3. Внесите изменения
4. Нажмите **«Сохранить»**

### 4.4. Автоматическое распределение

**Одиночное:**
1. Откройте карточку пациента со статусом «Ожидает распределения»
2. Нажмите **«Авто-распределить»**
3. Система подберёт отделение по правилам и отобразит результат

**Массовое:**
1. Перейдите в **«Очередь»**
2. Нажмите **«Распределить всех»**
3. Подтвердите действие
4. Система распределит всех пациентов из очереди

### 4.5. Ручное распределение

1. Перейдите в **«Распределение»** (верхнее меню)
2. Выберите пациента из списка ожидающих
3. Выберите отделение
4. Укажите причину назначения (обязательно для ручного распределения)
5. Нажмите **«Назначить»**

### 4.6. Очередь распределения

Раздел **«Очередь»** показывает всех пациентов со статусом «Ожидает распределения».

Для каждого пациента доступна кнопка **«Распределить»** для автоматического назначения.

### 4.7. История распределений

Раздел **«История»** показывает все выполненные распределения:
- Дата и время
- Пациент
- Из какого отделения / в какое отделение
- Кто назначил (врач)
- Тип (автоматическое / ручное)
- Причина

## 6. Распределение пациентов

### Алгоритм автоматического распределения

Система проверяет следующие критерии по порядку:

1. **Отделение активно** — не закрыто
2. **Свободные койки** — есть доступные места
3. **Пол** — правило и отделение допускают данного пациента
4. **Возраст** — правило и отделение соответствуют возрастной категории
5. **Диагноз** — основной диагноз пациента совпадает с профилем отделения
6. **Тяжесть психического состояния** — попадает в допустимый диапазон правила
7. **Тяжесть физического состояния** — попадает в допустимый диапазон правила

Приоритет правил: правило с наименьшим номером приоритета проверяется первым.

### Статусы пациента

| Статус | Описание | Действия |
|---|---|---|
| **Ожидает распределения** | Только что зарегистрирован | Авто- / ручное распределение |
| **Распределён** | Назначен в отделение | Просмотр, переназначение |
| **Госпитализирован** | Переведён из приёмного | Только просмотр |
| **Отказ** | Пациент отказался от госпитализации | Только просмотр |

## 7. Отчётность

### Экспорт в CSV

1. Перейдите на панель управления
2. Нажмите **«Экспорт в CSV»**
3. Файл загрузится с данными всех пациентов

### Поля экспорта

ФИО, Дата рождения, Возраст, Пол, Диагноз, Псих. тяжесть, Физ. тяжесть, Статус, Отделение, Дата поступления, Врач.

## Часто задаваемые вопросы

**В: Как добавить новое отделение?**
> Администратор → Справочники → Отделения → Добавить

**В: Что делать, если нет подходящего отделения?**
> Используйте ручное распределение или обратитесь к администратору для настройки правил.

**В: Как изменить распределение уже назначенного пациента?**
> Откройте карточку пациента → Авто- или ручное распределение (пациент будет переведён в новое отделение).

**В: Можно ли удалить пациента?**
> Да, через карточку пациента → «Удалить». Рекомендуется использовать статус «Отказ» вместо удаления.

**В: Как посмотреть загрузку отделений?**
> Панель управления → раздел «Загрузка отделений» или меню «Распределение → Отделения».
